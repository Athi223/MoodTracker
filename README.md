# MoodTracker

A simple React app to track your daily mood

<ul>
    <li>Track your daily mood</li>
    <li>Visualize how your week/month went by</li>
    <li>Monthly stats for an effortless overview</li>
    <li>Available in Light/Dark modes</li>
</ul>
